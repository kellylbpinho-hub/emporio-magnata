📋 Como usar o app, no dia a dia

Todo dia → aba Caixa: lance quanto entrou e quanto saiu — dá pra já lançar a baixa de estoque no mesmo lugar.
Toda vez que chegar carne → aba Estoque: registre o quanto entrou (dá pra escanear o código da nota).
Quando sobrar osso, apara ou perder peso → aba Estoque, seção "Perdas": registre pra não sumir sem explicação.
Quando cadastrar um item novo → aba Produtos: nome, custo e preço de venda (dá pra escanear o código de barras).
De vez em quando → aba Preço: pra descobrir por quanto vender um corte novo — já vem com os cortes mais comuns do açougue na lista.

Calculadora de Preço — qualquer item

Serve pra carne, bebida, artesanal, combo — qualquer coisa que você vende. Preencha os 4 quadros abaixo, na ordem — o app calcula o preço sozinho.
💡 Em palavras simples: o preço de venda precisa cobrir 4 coisas — (1) o que você pagou pelo item, (2) a fatia das contas fixas do mês, (3) os impostos e taxas que somem quando você vende, e (4) o lucro que você quer ganhar. Esqueceu algum desses 4? É aí que o dinheiro "some" sem você perceber.

1. Quanto custou o item

Ex: comprou 100kg na nota, pesou 98kg → coloque 98.
Ex: de 100kg com osso, sobram 87,9kg limpos → coloque 87,9.
Use pro que você compra pronto e não perde peso: bebida, embalagem, ingrediente de artesanal, etc.

2. Fatia das contas fixas

Aluguel, luz, água, internet — tudo que você paga todo mês, vendendo muito ou pouco.
O app divide as contas fixas por esse volume e soma no custo de cada unidade.

3. Impostos e taxas da venda

Isso aqui é o que "some" do preço na hora de vender — diferente do custo, que é o que você já pagou antes.

4. Quanto você quer lucrar

Escolhendo um corte da lista, o app já preenche a categoria e o nome pra salvar, mais abaixo. Se não for carne ou o corte não estiver aqui, deixe assim e escolha a categoria manualmente.
Ao escolher a categoria, o campo abaixo já vem preenchido com um lucro sugerido — mas você pode mudar.

☁️ Guardar cópia na nuvem (avançado, opcional)

Sem isso, o app já funciona normal e salva tudo neste aparelho. Só configure aqui se quiser acessar os mesmos dados de mais de um celular/computador, ou se peça pra alguém da equipe técnica ajudar a configurar.

1. Crie uma conta grátis em supabase.com (login com Google, sem cartão)
2. Crie um novo projeto (qualquer nome, ex: "emporio-magna")
3. No menu lateral vá em SQL Editor → cole o código abaixo → Run
4. Vá em Project Settings → API → copie a Project URL e a chave anon public
5. Cole os dois campos abaixo e clique em Conectar

Já tinha conectado a nuvem antes de julho/2026? Rode isso também no SQL Editor, uma vez só: alter table caixa add column estoque_ref_id text; alter table estoque add column codigo text; alter table estoque add column validade date;
